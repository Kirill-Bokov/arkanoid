# Arkanoid

Классическая аркадная игра Arkanoid, реализованная на TypeScript с использованием PixiJS.

Демо: https://kirill-bokov.github.io/arkanoid/

## Описание

Проект представляет собой реализацию механики Arkanoid:
- управление платформой
- физика отскока мяча
- разрушение блоков
- уровни с различной структурой

Цель — очистить поле от блоков, не потеряв мяч.

## Технологии

- TypeScript
- PixiJS
- Vite
- HTML + CSS

## Запуск локально
